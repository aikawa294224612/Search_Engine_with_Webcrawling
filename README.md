# Search_Engine_with_Webcrawling
使用Djangle製作搜尋引擎，可爬找最新Time和Yahoo News的top3新聞，並且可將熱門文字show成wordcloud和rank直條圖做視覺化

## Info.
Framework: Djangle
Visualization: D3.js
Functions: 
- Searching Engine
- TIMES Top 3 post
- Yahoo News Top 3 post
- Keyword Cloud
- Rank
